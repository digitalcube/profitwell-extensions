Simple tracking code inserter for Profitwell.

== Description ==

ProfitWell Retain needs you to add a bit of JavaScript to your application so that it can display in-app notifications to your users, as well as overlay our payment forms on top of your marketing site.

And the plugin can easy to add the snipet into your WordPress site.

== Installation ==

- Install the plugin
- Go to Settings > Reading page
- Put your Profitwell public api key
